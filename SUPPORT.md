# Support

This project is experimental and maintained as an open bench for citation-backed cloud pricing.

## Getting Help

Use GitHub issues for:

- Bugs in ingest, normalization, API behavior, citations, or the frontend.
- Reproducible setup problems.
- Provider pricing data issues with enough detail to reproduce.
- Documentation gaps.

Use GitHub Discussions if they are enabled for:

- Design questions.
- Provider support ideas.
- General usage questions.
- Comparisons of possible implementation approaches.

If Discussions are not enabled, open an issue and label it as a question if labels are available.

## Before Opening an Issue

Please include:

- The command or workflow you ran.
- Your Python version and operating system.
- Relevant provider, region, instance type, or query inputs.
- Whether `just check` passes.
- Any relevant error output.

Do not include secrets, `.env` contents, API keys, private traces, or provider account data.

## Security Reports

Security reporting is handled separately in `SECURITY.md`. Do not file public issues for vulnerabilities or suspected secret exposure once that policy exists.
